# Project 007: Number Guessing Game

**Short Description:**  
This project is a simple game where the computer selects a random number, and the user has to guess it.

**Skills Learned:**  
- Using the random module  
- Generating random numbers  
- Loops and conditional statements  
- User input and validation  
- Basic game logic

**How to Run:**  
1. Open the `number_guessing_game.ipynb` file in Jupyter Notebook.  
2. Execute the cells in order.  
3. Try to guess the number selected by the computer!
